## 1. Project Overview

- **App**: iOS (SwiftUI) memory game (shapes, numbers, letters)
- **Chain**: Polygon Amoy testnet
- **Goal**: reward contributors with *AutestCoin ($AUT)* and govern the project via a DAO

Right now we are in **early alpha**:
- Game + Web3 integration are under active development
- DAO + bounty system are being designed
